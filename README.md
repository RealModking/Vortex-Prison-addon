# Prison Addon

A custom Minecraft Bedrock Prison Addon focused on progression, mining, enchantments, economy systems, and scalable prison gameplay.

---

## ✨ Features

* ⛏️ Custom prison mine system in development
* 🪓 Custom player pickaxes in development
* 📊 Blocks mined tracking
* 💰 Economy & progression systems
* 🎟️ Key and token systems
* 💎 GemFinder enchantment
* 🔑 KeyFinder enchantment
* 💸 AutoSell enchantment
* ⚡ Tokenator enchantment
* 🏗️ Configurable mines
* 📦 GUI systems
* ⚙️ Server configuration support
* 🚀 Optimized for Minecraft Bedrock

---

## 🪓 Custom Pickaxe System

Every player gets a more personalized mining experience.

### Pickaxes include:

* Player name display
* Blocks mined statistics
* Custom progression systems
* Upgrade support

---

## ⛏️ Mine System

The addon includes a fully customizable prison mine system.

### Features:

* Multiple mines
* Mine progression
* Mine teleport GUI
* Auto reset support
* Configurable mine sizes
* Larger mines currently in development

## 🚀 Installation

1. Download the latest release
2. Import the `.mcaddon` file
3. Enable:

   * Holiday Creator Features
   * Beta APIs (if required)
4. Apply the pack to your world
5. Restart the world/server
